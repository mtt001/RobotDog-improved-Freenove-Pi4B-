Tutorial: 3 Coordinate Systems in Digital Twin

Orbit with mouse drag · Zoom with scroll · Use buttons to highlight each frame

1. Dog FLU Frame XYZ (body-fixed, labeled on model)

+X = Forward, +Y = Left, +Z = Up — right-handed (X×Y=Z)

2. Three.js Scene Frame xyz

+x = Forward, +y = Up, +z = Right — right-handed

3. Key Mapping

Dog X (Fwd) = Scene x (Fwd)
Dog Y (Left) = Scene −z
Dog Z (Up)  = Scene y (Up) ← this is why panel label is wrong!

4. Side-View Projection Plane

Camera looks along dog +Y (from right side). Visible plane = Dog X-Z = Scene x-y.
Panel should label axes as X horizontal, Z vertical (dog frame), not "X, Y".
